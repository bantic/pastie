== Name change
  As of July 23, 2012, this gem's name has been changed to 'bantic-pastie' so that
  the pastie.org organization can have the unqualified namespace for their own purposes.

= pastie

     $ gem install pastie
     ... copy something to your clipboard ...
     $ pastie
     Your pastie url is: http://pastie.org/private/xclxkcj. It has been copied to your clipboard.
